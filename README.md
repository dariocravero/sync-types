# sync-types

Sync types to use in your flux/redux actions

[![Build Status](https://travis-ci.org/dariocravero/sync-types.svg)](https://travis-ci.org/dariocravero/sync-types)

```
npm install --save sync-types
```

```js
var s = require('sync-types');

s.DOING //=> 'DOING'
s.DONE //=> 'DONE'
s.FAILED //=> 'FAILED'
s.TODO //=> 'TODO'
```
